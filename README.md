# wgnUtility
SimpleStyle第139回シャープの手書きメモWG-N10を極楽シンクロする『wgnUtility』

SimpleStyleは、ライフハックできるツールを<strong>実際につくり、使い、役立てるライフスタイルを紹介</strong>します。

<blockquote><big><strong>■ソフトの概要とそれでよくなること</strong></big></blockquote>

 Windows用: 『<strong><strong>wgnUtility</strong></strong>』はシャープの手書き電子ノート『<strong>WG-N10</strong>』と接続したときに、ワンタッチでデータをPCにバックアップし、PCで作成したメモを転送できるソフトです。

 『<strong><strong>wgnUtility</strong></strong>』を使うと、電子ノート『<strong>WG-N10</strong>』と簡単にメモをやりとりできるようになり、『<strong>WG-N10</strong>』とPCを組み合わせて活用できるようになります。


<blockquote><big><strong>■開発のアイデアとコンセプト</strong></big></blockquote>

 手書きメモ環境にずっと注目してきました。

 1991年以降、PCは電子ペンでしか使ったことがありません。

 1996年には携帯端末用の手書きメモソフトをリリース。

 2004年にはそれをブラッシュアップした『<strong>SmartWrite</strong>』でIPA未踏ソフトウェアに採択されました。

 それ以降も、『<strong>NetSticky</strong>』などで、手書きメモを容易にとれる環境を構築しています。



<blockquote><big><strong>■手書きメモ</strong></big></blockquote>

 アイデアを展開するのには、文字で文字でも自由に描ける手書きメモが最重要だと考えています。

 キーボードは、まとまったアイデアを形にしていくときには有効ですが、自由に広がるアイデアをフォローするのがむずかしいと感じているためです。

 携帯電話やスマートホンなどの小さすぎるディスプレイもアイデアを制約してしまうように感じます。

 梅棹忠夫がB6カードを使い、最近だと片岡義男がカードを使っているように、アイデアを出すのには、それなりの文章やスケッチを簡単に描けるメモ用紙が重要だとわたしも考えます。

 じっさいに、『<strong>SmartWrite</strong>』プロジェクトでは、を作成したときには、3週間で497枚、6カ月で1,188枚のメモを『<strong>SmartWrite</strong>』で書きました。これはなかなかの規模で、それを紙のようにめくってブラウジングするソフトも作りましたから、もう手書きメモは『<strong>SmartWrite</strong>』でいい、とさえ思ったこともありました。



<blockquote><big><strong>■『<strong>SmartWrite</strong>』から『<strong>NetSticky</strong>』へ</strong></big></blockquote>

 残念ながら、『<strong>SmartWrite</strong>』を実装したWindowsXPとVAIO U（5型で800×600ピクセル）の特にバッテリー関係の寿命が尽きたこと、.NET1.1からの拡張性の課題に直面したこと、外注でプログラムを頼んでいたのですがそれができなくなったことなどから、新しい環境に対応することがむずかしくなりました。

 そこで捲土重来、プログラムを修得、自分でゼロから書いたのが『<strong>NetSticky</strong>』です。ほかにAndroid版を試作したりもしています。

 バッテリーの問題から、使う機会はめっきり減ってしまいましたが、最近タブレットが増えてきているので、復活の時を、虎視眈々とねらっています。



<blockquote><big><strong>■課題</strong></big></blockquote>

 さまざまなソフトをいくつも作ってきて感じている課題は、おおきく3つあります。

（1）適当なハードウェアに実装できること

（2）操作がたやすいこと、標準形式であること、互換性シンクロがたやすいこと

（3）画面と描ける文字のバランスがよいこと

です。

 1996年は専用端末、のちにWindowsXP/7やAndroidに実装した経験からいうと、メモ専用のハードウェアの方が、電池の持続時間などが優れていることが多い、ということがいえます。最近のタブレットは動作時間も長くなってきましたが、とっさにさっと出して描ける点では、専用端末は魅力的です。

 操作や標準形式や互換性は、手書きメモのように、あまり高度なことをしないアプリケーションにとっては、長期にわたって使う観点からも重要です。最近だと、Dropboxなどのシンクロ環境と融合できるとよいと思います。ファイル形式はExifまたはpngが望ましいでしょう。

 画面ですが、梅棹忠夫がB6を、片岡義男が5×3インチを押すように、わたしは5型程度のディスプレイを押します。できればペンタイプが望ましいです。指で描く場合には、もっと大きなサイズが必要になります。指では1文字のサイズは7×7cm以上にもなってしまうからです。これでは、何文字かしか書くことができず、メモの体をなさないのです。

 タッチペンなら、1文字のサイズは2×2cm以下で、5型のディスプレイでだいたい50文字くらいは書くことができます。紙と異なり、描ける文字にはだいぶ制約がありますが、それでも50文字くらい書ければ、それなりの文章にはなります。

 Android版を試作しただけでリリースしなかった大きな理由は、この指の分解能と画面のサイズのアンバランスを解消できなかったためです。既存の手書きメモには、描くところと見るところを分けるなどの工夫をしているものもありますが、一般的に言えばそのようなことをすると操作が複雑になって、簡単にメモを取るというそもそもの目的を逸脱しがちです。



<blockquote><big><strong>■理想の手書きメモツール</strong></big></blockquote>

 こんなバックグラウンドをもつ筆者は、理想の手書きメモツールをいつも楽しみにしています。現在は、ドキュメントスキャナの性能が高いことから、紙に描いてスキャンして保管、というのがもっともバランスがよいと考えていますが、ハードウェアごと設計する機会があれば、『<strong>NetSticky</strong>』を実装したいと考えています。



<blockquote><big><strong>■シャープ『<strong>WG-N10</strong>』登場</strong></big></blockquote>

 さてそこで、シャープの『<strong>WG-N10</strong>』です。

 タッチペン（爪や指でも可能）+5型メモリ液晶（800×600ピクセル）、約30日動作、本体のみで210グラムと、最小のハードウェアで実装した手書きメモ専用機です。書き換え時のみ電力を使用するメモリ液晶という種類らしく、16階調モノクロです。じつにしぶい。シャープというよりは、むしろキングジムやカシオあたりから出てきてもおかしくない感じの商品です。

 上記のように、わたしは手書きメモに関してはプロの（あるいは上級の/目の肥えた/注文の多い）設計者/研究者/利用者であり、なおかつ『<strong>NetSticky</strong>』はカラー版があることからもおわかりいただきたいのですが、このシャープの『<strong>WG-N10</strong>』は、わたしにとってはストライクゾーンからだいぶ外れる商品です。

 しかし、ここは重要です、ストライクゾーンから外れようが外れまいが、手書きメモ専用機なんてのは、ごく少ないのです。それがあるだけでも評価すべきなのです。

 試用したところでは、メニューはPCとの連携でやや操作が剥き出しな感じはしますけれども、全体的には古きよきガジェット的な印象で、この手のツールを求めるユーザーにとっては、なじみの感じです。ずばり、2,980円なら買いでしょう。え、14,800円…。う～む。この一文、削除…ですかね。まあ価格の認識はさておき…。



<blockquote><big><strong>■テンプレート</strong></big></blockquote>

 このシャープの『<strong>WG-N10</strong>』に欠けているものはふたつあります。

 ひとつはテンプレート。もうひとつはPCあるいはインターネットやクラウドとの連携機能です。

 筆者が『<strong>SmartWrite</strong>』を作ったときに、もっとも注力したもののひとつが、テンプレートでした。それは、ユーザーは自分の好きな背景を見ていれば、機能のよしあしにかかわらず、ソフトへの愛着をもつと考えたためです。

 『<strong>WG-N10</strong>』には、テンプレートの追加機能がありません。PCからの画像の転送機能はあるのですが、テンプレートは横経線、縦経線、方眼紙、ToDoなど9つのお仕着せから選ぶ以外には方法がありません。これはとても残念なことです。

 もっとも、テンプレートとして使用する画像を、たとえば10枚単位でPCから『<strong>WG-N10</strong>』に転送すれば、その10枚の範囲ではテンプレートとして使用できますので、アイデア次第で回避できなくはない気もします。



<blockquote><big><strong>■連携</strong></big></blockquote>

 ふたつめの連携機能です。

 『<strong>WG-N10</strong>』には、標準的なPCとの連携機能は存在しています。ただ、それはメニューをたどったり、ボタンを押したりする旧態依然としたもので、21世紀の水準からすると、やはりだいぶもの足りない気がします。

 とくに気になったのは、先の文字のサイズにも関係することですが、人間は読める文字のサイズと書ける文字のサイズが異なること、そしてPCと連携するのなら、キーボードやテキストを扱えればよいのに、『<strong>WG-N10</strong>』は、実直なほど画像というかビットマップというか、ピクセル画像しか扱えない点です。

 本稿は、べつに『<strong>WG-N10</strong>』はダメという観点で書いているものではありません。ダメなところがあるのなら、できるだけそれをサポートしちゃおうというのが筆者のモットーですから、連携機能（およびテンプレート追加機能）をできるだけ簡単にサポートするソフトを作りました。

 『<strong><strong>wgnUtility</strong></strong>』です。



<blockquote><big><strong>■バックアップと転送を省力化</strong></big></blockquote>

 『<strong><strong>wgnUtility</strong></strong>』にはふたつの機能があります。

 ひとつは、シャープの手書き電子ノート『<strong>WG-N10</strong>』と接続したときに、ワンタッチでデータをPCにバックアップする機能です。ファイルはpngに変換します。

 もうひとつは、PCで作成したメモやテンプレートを転送する機能です。

 いずれも[execute]ボタンひとつで動作します。



<blockquote><big><strong>■使い方</strong></big></blockquote>

（1）exeとおなじフォルダにあるpngファイルがテンプレートファイルです。サイズがおなじであれば任意のデザインを使用できます。好きなソフトで凝りまくって作ってください。最終的に転送できるのはモノクロ16階調になりますが、テンプレートにはpngである以外の制約はありません。

（2）『<strong><strong>wgnUtility</strong></strong>』を起動します。

（3）『<strong>WG-N10</strong>』を接続します。

（4）root pathに、『<strong>WG-N10</strong>』のEXPORTを指定します。自動認識するはずなので、指定しなくてもかまいません。

（5）target pathは、『<strong>WG-N10</strong>』から転送したデータを保持するフォルダです。標準ではアプリケーションを起動したフォルダになっています。

（6）もしも『<strong>WG-N10</strong>』に転送したいテキストがある場合、テキストを書いてください。ToDoとかが向くと思います。

（7）[execute]ボタンを押します。

 これで完了です。

 『<strong>WG-N10</strong>』でEXPORTしたデータがPCに入ります。

 テキスト枠に書いた文字をビットマップ化した画像が『<strong>WG-N10</strong>』に入ります。文字を書かなければ、単なるテンプレートとして転送します。



<blockquote><big><strong>■ワンポイント</strong></big></blockquote>

 原稿を書いていて気づいたのですが、テンプレートのみを一括して10ページ分とか転送する機能があるともっとよかったかもしれませんね。

  『<strong><strong>wgnUtility</strong></strong>』は、Windows用のツールです。
